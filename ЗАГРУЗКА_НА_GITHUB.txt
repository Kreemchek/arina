❌ "Repository not found"
   → Создайте репозиторий на GitHub сначала!
   → https://github.com/new

❌ "Authentication failed"
   → Используйте TOKEN, а не пароль!
   → Token начинается с: github_pat_...

❌ "Permission denied"
   → Проверьте права токена
   → Токен должен иметь права на repo

❌ "./push_to_github.sh: Permission denied"
   → Выполните: chmod +x push_to_github.sh
